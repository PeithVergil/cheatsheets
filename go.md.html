**Go** Installation ============================================================ Download the Go SDK. ~~~ bash wget https://dl.google.com/go/go1.11.linux-amd64.tar.gz ~~~ Extract the archive. ~~~ bash tar -C /usr/local -xzf go1.11.linux-amd64.tar.gz ~~~ Add to system path. ~~~ bash export PATH=$PATH:/usr/local/go/bin ~~~ Usage ============================================================
